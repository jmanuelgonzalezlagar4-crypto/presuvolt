Política de privacidad de PresuVolt

Última actualización: 23 de julio de 2026

Qué es PresuVolt

PresuVolt es una aplicación para instaladores que permite crear presupuestos y partes de trabajo mediante notas de voz.

Qué datos se recogen

El desarrollador de PresuVolt no recoge, no almacena y no vende ningún dato personal de los usuarios. La aplicación no contiene publicidad ni rastreadores de terceros.

Dónde se guardan tus datos

Los datos que introduces (clientes, presupuestos, partes de trabajo, precios y fotografías) se guardan en tu propio dispositivo. Si activas la sincronización en equipo, los datos se guardan además en una base de datos de Firebase (Google) creada y controlada por ti, a la que solo se accede con tu código de empresa. El desarrollador no tiene acceso a esa base de datos.

Reconocimiento de voz

El dictado por voz utiliza el reconocimiento de voz del propio dispositivo (Google). El audio se procesa según la política de privacidad de Google. La aplicación solo recibe el texto resultante y no almacena las grabaciones.

Inteligencia artificial (opcional)

Si introduces voluntariamente tu propia clave de la API de Anthropic, el texto de tus dictados se envía a la API de Anthropic para procesarlo. Esa clave se guarda únicamente en tu dispositivo. Sin clave, la aplicación funciona sin conexión con un analizador local.

Permisos

La aplicación puede solicitar acceso al micrófono (para dictar), a la cámara o galería (para adjuntar fotos a partes y presupuestos) y a los contactos (opcional, para rellenar datos de clientes). Estos datos nunca salen de tu dispositivo o de tu propia base de datos de sincronización.

Eliminación de datos

Puedes borrar todos los datos desinstalando la aplicación y, si activaste la sincronización, eliminando tu proyecto de Firebase.
